A ideia dessa cifra de codificação é receber um array diverso,

EX: A B C 1 2 3 4

Encontrar seu valor em ASCII

65 66 67 48 49 50 51

Então criamos blocos ASCII para desenhar

DETAILS

512 - MAX characters for the cypher on a 32x32 image;

color match example:

0 BLUE = X
1 RED = O
2 GREEN = L
3 PURPLE = Z
4 ORANGE = C
5 BLACK = V
6 WHITE = B
7 BROWN = M
8 PINK = Q
9 YELLOW = W
NULL (AVOID) - GRAY = E

steps of conversion:
GET TEXT -> CONVERT INTO ASCII (INT) -> CONVERT EVERY ASCII INTO A BIG ASS STRING
GET TOTAL MESSAGE  SIZE -> CUT AT 1024 CHARS, ELSE FILL WITH  GRAYS.
DRAW 32X32 IMAGE RESEMBLING THE COLOR BLOCK, SAVE THE STRING.
WRITE A .XPM FILE WITH THE CODE COLOR BLOCK, AND ATRIBUTE VALUES CORRESPONDING TO THE CYPHER PASSED

steps of deconversion: 
READ A XPM FILE AND FETCH VALUES ACCORDING TO THE DECRYPTO CYPHER.
WE WILL NEED TO DISMANTLE A XPM, WHICH MAY BE EASY.
